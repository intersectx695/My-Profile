# Quick Start Guide

## First Steps

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Update Your Profile Data**
   - Open `data/profile.ts`
   - Replace all placeholder content with your actual information:
     - Personal details (name, title, bio)
     - Professional experiences from your resume
     - Education details
     - Achievements, patents, awards
     - DataGenie information
     - Contact information (email, LinkedIn, etc.)

3. **Test Locally**
   ```bash
   npm run dev
   ```
   Visit http://localhost:3000

4. **Build for Production**
   ```bash
   npm run build
   ```

## Deployment

### Option 1: Vercel (Easiest)
1. Push code to GitHub
2. Go to [vercel.com](https://vercel.com)
3. Import your GitHub repository
4. Deploy (automatic)

### Option 2: Netlify
1. Push code to GitHub
2. Go to [netlify.com](https://netlify.com)
3. Import your GitHub repository
4. Deploy (automatic)
